# strong-dashboard

# Cardiovascular Risk in Transmasculine Individuals

## Overview
This dashboard presents findings from the STRONG cohort, examining acute cardiovascular risk in transmasculine individuals relative to matched cisgender male and female referents across Kaiser Permanente health systems.

## Project Importance
Transmasculine individuals face elevated VTE and stroke risk relative to cisgender women, consistent with partial masculinization of vascular risk. These findings support the need for long-term cardiovascular monitoring in transmasculine persons receiving gender-affirming hormone therapy.

## Dashboard
Live dashboard: https://li4oxd-applecare11s0s-smith.shinyapps.io/DASHBOARD/

## Data
- Source: STRONG cohort, Kaiser Permanente (2006–2024)
- N = 345,158 individuals
- Outcomes: VTE, ischemic stroke, myocardial infarction
